Find K Closest Elements

Two-pointer shrinking window — visualize every comparison and drop

Core idea: Start with the full array window. At each step, compare the left and right elements' distances from x. Whichever is farther (or equal — prefer keeping left) gets dropped. Repeat until the window has exactly k elements.
